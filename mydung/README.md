# My Dung

This is a distributed system project in Go (Golang) called "My Dung". This project is a starting point for building your own distributed system experiments and prototypes.

## Structure
- `main.go`: Entry point for the application.
- `cmd/`: (Recommended) For command-line tools.
- `internal/`: (Recommended) For private application and library code.
- `pkg/`: (Recommended) For public libraries.

## Getting Started
1. Make sure you have Go installed.
2. Run the project:
   ```sh
   go run main.go
   ```
